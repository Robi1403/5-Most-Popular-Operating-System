WINDOWS ORIGIN

The history of Windows dates back to 1981 when Microsoft started work on a program called "Interface Manager." Few years later, founder of Microsoft 'Bill Gates' announced for Microsoft Windows on the 10th of November 1983 and released the first version of Windows in 1985. It was designed to run on top of a command-line operating system MS-DOS.

Windows heavily relies on a graphical user interface (GUI), first developed by Xerox in 1975. A conventional GUI will consist primarily of the combination of “window, icon, menu and pointing device”, or WIMP.

Windows 1.0 (1985)

Windows 1 is the first Windows operating system which was released in 1985. In the development of this operating system, Microsoft attempted to create a graphical user interface where users can click on icons and buttons on the screen instead of using command prompts. Windows 1 was a mouse dependent OS which was not common at that time.

Windows 2 (1987)

Released in December 1987 Windows 2 introduce the feature of program overlapping, making multiple program run simultaneously and switching between the programs was easier. Features included in this version are the feature to maximize and minimize windows, a control panel to find system settings and options, and the release of Microsoft word and excel.

Windows 3 (1990)

The first version of Windows that became popular was Windows 3 that was released in 1990. It came pre-installed on many PC-compatible computers. One of the well-known features of Windows 3 is that it allows users to run multiple programs simultaneously, which is also known as multitasking. It also supports 256 colors making a more modern and colorful looking operating system. Another feature of it was the game Solitaire, which was a big hit with users. Windows 3 sells more than 10 million copies making it one of the best-selling GUI ever.

Windows 3.1 (1992)

Windows 3.1 was released in 1992 and served as an upgrade to Windows 3.0. It was the first Windows operating system that became a good publishing platform as it has TrueType fonts. This operating system required 1MB of RAM, was the first Windows OS to be distributed on a CD-ROM, allowed Microsoft DOS programs to be controlled with a mouse, and introduced Minesweeper.

The major changes of Windows 3.1 from its previous version is its better multimedia and networking support, better error handling, and TrueType font support. This version has sold three million copies in just two months.

Windows 95 (1995)

Three years later, Windows 95 was released along with its new features. The operating system comes with different changes but the most noticeable was the Start Menu and Start button, and also introduced the “plug and play” feature that was supposed to find and install new drivers for peripherals automatically. It also focused on multitasking and had a 32-bit environment.

Windows 95 was designed to operate with Windows NT and to have a greater backward compatibility with previous drivers and softwares. It also introduced the Internet Explorer yet not installed by default.

Windows 98 (1998)

Windows 98 was released in 1998 as an update to Windows 95. It comes with a bunch of new features and improvements such as enhanced support for USB devices, bigger hard drives, and as well as user interface changes.

Windows 98 has the ability to use larger hard drives and files, thanks to the new file system called FAT32. The OS also comes with new programs, such as Internet Explorer 4, Outlook Express, and Windows Media Player.

Windows ME (2000)

Windows ME released in 2000 it was the Millennium edition of the Microsoft Operating system. It was the final version of Windows designed for home use and based on MS-DOS. Windows Media Player 7, Windows Movie Maker, Internet Explorer 5.5, and automated system recovery capabilities were among the new features that Windows ME brought.

It was known for its errors and installation issues, however. Windows Me is based on Windows 98.

Windows 2000 (2000)

Windows 2000 was released in February which was based on Windows NT. It was the first operating system to support hibernation and it also comes with Microsoft's automatic updating. The operating system comes with three different varieties: Professional, Server, and Advanced Server.

It offered a lot of improvements such as the higher RAM support that goes up to 4GB, stronger internet integration, eliminated many reboot scenarios, and the Windows FIle Protection.

Windows XP (2001)

Released in 2001 Windows XP, combined the best features of Microsoft's enterprise and consumer lines of operating systems. The Start menu, taskbar, Vista wallpaper, and additional visual effects were all updated in its UI. It also brought it new features including integrated CD burning, ClearType, auto-updates, and functional recovery tools. Despite its long career and popularity, it was vulnerable to hacking and cyberattacks due to security weaknesses, especially with Internet Explorer.It had several versions, including Home and Professional, and new editions like Media Center, Tablet PC, and Professional x64.

Windows Vista (2007)

On November 8, 2006, Windows Vista went into manufacturing and on January 30, 2007, it was made available through the Windows Marketplace. It had a fresh look, with transparent elements and a focus on search and security.

However it had a few issues. Users had to approve every app that wanted to make changes this led to carelessness with security. Windows Vista brought new features like Windows Aero, faster search and multimedia tools, and security improvements, but it was also slow on older computers even though it was advertised as "Vista Ready."

Windows 7 (2009)

Windows 7 was released in 2009 as the predecessor to Windows Vista. It was regarded as an important improvement since it aimed to fix users complaints and problems with Vista. Windows 7 was quicker, more stable, and easier to use, and many people upgraded to it from Windows XP, skipping Vista entirely. Handwriting recognition and the ability to snap windows to the sides of the screen for easier resizing were two of Windows 7's new features. Windows 7 has five versions Starter, Home Premium, Professional, Enterprise, and Ultimate.

Windows 8 (2012)

Windows 8 was released in 2012 and comes with a new interface with the Start Menu and the Start Button that was replaced with a touch-friendly Start screen. The new interface had icons and widgets. Despite its new look, a desktop interface similar to Windows 7 was still available.

The operating system was faster than previous versions and has support for USB 3.0 devices. It also introduced Windows Store, which offered universal Windows apps.

Windows 8.1 (2013)

The criticisms received from Windows 8, led to the creation of Windows 8.1, a free update released in 2013. The operating system had returned the Start Button back, which was missing from its previous version, and the option to boot directly into the desktop view.

Windows 10 (2015)

Microsoft released Windows 10 in July 2015 as a follow-up to Windows 8. It returns the Start menu and balances traditional desktop computers and touch-based devices like tablets. Universal apps may be installed through the Windows Store and operate on any Windows device thanks to Windows 10's platform-unifying design. It also allows you to convert between tablet, keyboard, and mouse modes on devices. Windows 10 has 6 variants Windows 10 Home, Windows 10 S, Windows 10 pro, Windows 10 enterprise, Windows 10 Education, and Windows IOT each have unique features.

Windows 11 (2021)

Windows 11, the latest version of Windows OS, was released in late 2021. It comes with new features such as the new look and feel of the interface, a new widget system, and the feature that can snap windows into different layouts. The default applications have been updated with new design settings that offer easier on changing system settings. Multiple desktop features were also available, making it a great help for organizing user’s work more efficiently.

Conclusion

Windows has come quite a long journey ever since it was first released in 1985. Microsoft offered new features and improvements to every version, in response to user feedback and new technologies. From Windows 1 to the latest version which is Windows 11, Microsoft’s operating systems have shaped the way we use computers today.
